## Descripción
Programa hecho en Java que permite registrar usuarios mediante un menu, en consola. 

El sistema permite registrar nombre, edad, correcto electronico y salario mensual del usuario. Implementa validaciones usando excepciones personalizadas. 

Para ejecutar el programa se necesita un editor de codigo compatible con Java, ademas de tener Java instalado. Clonar el repositorio y compilar el proyecto.

## Tecnologías utilizadas
- Java
- JDK 25
- Visual Studio Code
